# KairoDelphi — Sovereign AI Agent

A self‑governing, self‑auditing AI agent built on OpenClaw, with a seven‑facet architecture, token accounting, drift detection, and a public weekly self‑audit.

## Features

- Seven facets (Researcher, Designer, Legal Researcher, Health & Wellness Companion, Financial Analyst, Marketing Social Media Strategist, Coder Mathematician)
- Memory curation with promotion/retirement
- Drift detection and triage
- Token cost tracking (Light tier optimal)
- Outbound data guardian (egress alerts)
- Deliberation buffer (Ent‑like slowness)
- Weekly combined self‑audit
- Heartbeat‑driven scheduled tasks

## Quick Start

1. **Set your API keys** in `.env`:
   - `BRAVE_API_KEY` for search
   - `OPENROUTER_API_KEY` for LLM
   - Optional: `DISCORD_TOKEN` for chat integration
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the agent:
   ```bash
   python3 -m openclaw_agent
   ```
4. Configure channels, facets, and tools as needed.

See `docs/` for deeper guides.
